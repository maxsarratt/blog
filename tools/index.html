打字🔗

  • Code Typing Script - 用代码测试你的打字速度，可以选语言比如 js,go
  • 用小说内容来练习打字

  • Feedadd - 众包的形式运行爬虫机器人，上传数据

转换🔗

  • 百度文库转 word 或 pdf - 有瑕疵，勉强能用。
  • 微友活码 - 用微友活码生成的营销二维码，具有不受个人号加人数量限制，不受扫码进群百人限制，群二维码过期灵活更新，扫码数据实时统计等优点，是个人号、微群及企业微信私域流量沉淀，营销引流必备的免费工具！
  • Epub to HTML

Server🔗

  • Nginx Playground - 在线调试 nginx 服务器

Developmenets🔗

  • Docker Run - 使用 docker 来运行不信任的代码，运行后删除
